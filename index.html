Connect Your Google Sheet via Apps Script

Admin enters data in Google Sheets → GAS serves as backend → Dashboard updates live.

1
Set up your Google Sheet with these column headers in Row 1 (order flexible):
Date  |  Devotees  |  Chanting  |  Narasimha Kavach  |  Tulasi Parikrama  |  Tulasi offered
Each row = one entry. Numeric columns = counts.
2
Extensions → Apps Script → Paste the Code.gs content → Save → Deploy → New deployment → Type: Web app → Execute as: Me → Access: Anyone → Deploy → Copy the URL.
3
Paste the web app URL above and click 🔄 Refresh. Dashboard auto-refreshes every 5 minutes. ✨
